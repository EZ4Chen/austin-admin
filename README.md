## 快速开始

基于 [amis](https://github.com/baidu/amis) 渲染器，快速搭建自己的管理系统。

```bash
# 安装依赖
npm i

# 打开服务
npm start

# 线上可使用 pm2 部署启动
pm2 start server.js

```
